Introductie tot HTML

Wat is HTML?

HTML staat voor HyperText Markup Language. Het is de standaardtaal voor het maken en structureren van webpagina's. HTML gebruikt verschillende elementen (tags) om de inhoud van een webpagina te definiëren. Deze elementen geven de browser informatie over hoe de inhoud moet worden weergegeven.

Hoe gebruik je HTML?

Om een HTML-document te maken, gebruik je een teksteditor zoals Kladblok (Windows), TextEdit (Mac) of een gespecialiseerde code-editor zoals Adobe Dreamweaver, Visual Studio Code, Sublime Text, etc.

Een basisstructuur van een HTML-document ziet er als volgt uit:

			
				<!DOCTYPE html>
				<html>
				<head>
					<title>Titel van de pagina</title>
				</head>
				<body>
					<h1>Dit is een kop</h1>
					<p>Dit is een alinea</p>
				</body>
				</html>

In dit voorbeeld zie je de basisstructuur met de doctype-declaratie, de html-, head- en body-elementen. Tags zoals <h1> en <p> worden gebruikt om respectievelijk koppen en alinea's te maken.
