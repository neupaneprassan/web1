Wat is CSS?

CSS staat voor Cascading Style Sheets. Het is een opmaaktaal die wordt gebruikt om de presentatie (zoals lay-out, kleuren, lettertypen, enz.) van een HTML-document te controleren. Met CSS kunnen webontwikkelaars de vormgeving van elementen op een webpagina aanpassen en stijlen toepassen om de uitstraling van de inhoud te verbeteren.

Hoe gebruik je CSS?

Er zijn verschillende manieren om CSS toe te passen:

  1. Externe stijlbladen: Een apart CSS-bestand maken met de extensie .css en dit bestand koppelen aan het HTML-document met behulp van het <link>-element.
    Deze methode is het meest aangewezen bij het creëren van een site.
  2. Interne stijlbladen: Stijlen toevoegen binnen de <style>-tags in de <head> van het HTML-document.
    Deze methode gebruik je uitzonderlijk als de stijlen enkel en alleen voor dàt HTML-document gelden.
  3. Inline-stijlen: Stijlen rechtstreeks toevoegen aan HTML-elementen met het 'style'-attribuut. Dit wordt hoofdzakelijk en bij voorkeur aangewend door andere programmeertalen, zoals Javascript, om een (tijdelijke) stijl toe te voegen aan een HTML element in een bepaalde situatie. Een webdesigner gebruikt deze methode best niet, of hoogstens bij uitzondering, wanneer het design wordt omgezet in HTML/CSS.

Hier is een voorbeeld van het gebruik van een interne stijlblad:

				
					<style>
						body {
							font-family: Arial, sans-serif;
							color: #333;
						}
						/* Voeg meer stijlen toe */
					</style>
